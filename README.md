# Honey Badger AI 🦡

Honey Badger AI is a premium chatbot built using Streamlit and NVIDIA Nemotron API.
It supports real-time AI responses, PDF-based question answering, persistent chat history, and a premium glassmorphism interface.

## Features

* Real-time streaming AI responses
* PDF upload and document questioning
* Chat history storage
* Premium custom UI with animated styling
* NVIDIA Nemotron powered inference

## Tech Stack

* Python
* Streamlit
* OpenAI SDK
* PyPDF2
* Joblib

## Deployment

Deployed using Streamlit Community Cloud.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
